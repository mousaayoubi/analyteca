# Analyteca

Headless Magento Analytics & Operations Dashboard

Monorepo structure:

apps/api  → Node.js (Express + JWT + Postgres + Cron)<br />
apps/web  → React (Vite + Tailwind)<br />
magento   → Magento 2 module source (Analyteca_Insights)<br />
infra     → Docker & deployment configs<br />
docs      → Architecture diagrams & screenshots<br />
